# To-Do-List

## Tech Stack Used : 
1. EJS,Bootstrap,CSS
2. jQuery
3. Express.js
4. PostgreSQL

## Commands to run 
1. npm init -y
2. npm i ejs body-parser express pg nodemon
3. nodemon app.js (For live local hosting)
